# 車種別部品調達原価一覧

## 概要
本ドキュメントは、2022年1月～2025年12月の期間における、各車種の部品調達原価（1台あたり）をまとめたものです。BOMマスタに基づく直接材（エンジン、トランスミッション、ECU、内装部品、サスペンション等）の調達コストを集計しています。

- **総レコード数**: 11,264件
- **対象期間**: 2022年1月～2025年12月
- **平均1台あたり原価**: ¥2,001,730

### 考察
- 間接材は直接材の**約10%**の規模で、工場運営に必要なMRO資材（保守・修理・運用）を構成
- 安全保護具が間接材の**25%**を占め、工場安全管理への投資が最大
- 測定工具・切削工具などの生産設備関連で**約28%**、製造品質維持に重点
- 平均調達額は直接材が間接材の**約3.8倍**で、直接材は高額部品中心、間接材は少額多品種

## データ生成仕様

### 調達タイミング
- 受注日と同日に調達発注
- リードタイム: 7日後に納品予定
- 実際の納品: 予定日 ±5日

### 拠点マッピング
- KMM（熊本製作所） → STM（埼玉製作所）
- HTB（ホンダオートボディー） → HTB
- SZK（鈴鹿製作所） → SZK
- HMM（浜松製作所） → SZK

### 数量計算
- 調達数量 = 受注数量 × BOMの`component_quantity_per`
- 例: FIT-GR3を5台受注 → 鋼板材料は 5台 × 2個 = 10個調達

### サプライヤー
ダミーサプライヤー5社からランダム選択:
- SUP-001: 日本製鉄
- SUP-002: 住友電気工業
- SUP-003: デンソー
- SUP-004: アイシン
- SUP-005: 矢崎総業

## 注意事項

1. **直接材のみ**: このデータは直接材（製造に直接使用される部品）のみを含みます
2. **間接材除外**: MRO資材、消耗品、オフィス用品等は含まれません
3. **税込価格**: すべての金額は消費税10%込み
4. **配送料含む**: 配送料・手数料を含んだ総コストです
5. **割引考慮**: 一部レコードには割引が適用されています（最大5%）
